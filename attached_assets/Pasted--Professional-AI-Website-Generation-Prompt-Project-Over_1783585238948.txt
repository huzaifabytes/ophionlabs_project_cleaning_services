# Professional AI Website Generation Prompt

## Project Overview

Create a **premium, modern, flat, responsive cleaning services website** for small to medium cleaning businesses.

The design should feel premium, trustworthy, elegant, and conversion-focused.

**Theme**

- Modern
- Flat Design
- Light Theme
- Premium UI
- White background
- Soft shadows
- Rounded corners (12–16px)
- Spacious layout
- Smooth animations
- Glassmorphism only where appropriate (very subtle)
- Fast loading
- Fully responsive (Desktop, Tablet, Mobile)

Color Palette

- Primary: #0074FC
- Secondary: #000C26
- Background: #FFFFFF
- Accent: #F6F9FC

Typography

- Clean modern font (Inter, Poppins, or Manrope)
- Large headings
- Comfortable spacing
- Professional hierarchy

# Google Sheets Settings

The Google Sheet configuration must NOT be hardcoded.

Instead,

Create a dedicated Google Sheets settings page inside the Admin Panel.

Admin can update

- Google Spreadsheet ID
- Sheet Name
- Google Apps Script URL
- API configuration (if applicable)
- Authentication details (if required)

Admin should be able to switch to a completely different Google Sheet at any time without modifying code.

All future form submissions must automatically use the newly configured Google Sheet.
